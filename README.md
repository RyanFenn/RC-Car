# RC-Car
A repository for the RC Car project. 

**Project Summary:**
This project uses as Raspberry Pi to control the RC Car over Wi-Fi. The car as a camera module that is used to stream video to a browser on a computer. The RC car is intended to be controlled using a computer with a keyboard. The following keys can be pressed to control the car: 
- "W" -> forward
- "S" -> reverse
- "D" -> hard right
- "A" -> hard left
- "W+D" -> soft right
- "W+A" -> soft left
- no keys pressed -> stop
- "+" -> increase speed
- "-" -> decrease speed
